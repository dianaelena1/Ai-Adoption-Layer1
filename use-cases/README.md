# Use Cases - AI in Practice (Layer 1)

Aici fiecare participant isi pune use case-ul lui de AI, pe care il va prezenta la final.

## Cum adaugi use case-ul tau

1. Copiaza fisierul `_TEMPLATE.md`.
2. Redenumeste-l cu numele tau, ex: `ion-popescu.md`.
3. Completeaza cele 5 sectiuni (taskul, prima incercare, ce a mers/ce nu, ce ai schimbat, rezultatul).
4. Urca fisierul in acest folder (vezi cele doua variante mai jos).

## Conventie de denumire
- Un fisier per persoana: `prenume-nume.md`
- Daca ai mai multe fisiere (capturi, cod), fa-ti un subfolder: `use-cases/prenume-nume/`

## Variante de a urca fisierul

### Varianta A - direct in browser (cea mai simpla, fara git)
1. Deschide repo-ul pe GitHub, intra in folderul `use-cases`.
2. Click pe **Add file -> Create new file** (sau **Upload files**).
3. Scrie numele fisierului si continutul, apoi **Commit**.

### Varianta B - cu git (daca esti colaborator)
```
git clone https://github.com/dianaelena1/Ai-Adoption-Layer1.git
cd Ai-Adoption-Layer1/use-cases
# creezi fisierul tau, apoi:
git add .
git commit -m "Add use case - Ion Popescu"
git push
```

> Daca nu poti urca (nu ai drepturi), cere-i lui [Diana] sa te adauge ca **collaborator**
> pe repo, sau trimite-i fisierul si il urca ea.
